
Hi\_App
================

A Ruby gem to create a sinatra hello world app. Just to use for testing.

Installation
------------

    gem 'Hi_App'

Usage
------

In your shell:

    Hi_App create app_name
    

Run Tests
---------

    git clone git@github.com:da99/Hi_App.git
    cd Hi_App
    bundle update
    bundle exec bacon spec/main.rb
